## 一、最重要的事：外链策略

**小红书对站外引流严格限流，正文出现 URL 会直接压曝光，甚至判违规。** 处理方式按优先级：

1. **正文完全不放链接**，图片里也只放「搜索 human-tokenizer」
2. 发布后**自己在评论区发一条纯文本**：`github 搜 human-tokenizer，或者浏览器输 leesieg.github.io/human-tokenizer` 并置顶
3. 个人简介里放链接（简介外链相对安全）
4. 如果笔记爆了，评论区会有人问「链接呢」——及时回复，回复的互动还能助推流量

## 五、发布策略

- **时间**：工作日 20:00–23:00，或周日晚（打工人 emo 高峰期，共鸣内容效果最好）
- **话题标签**：8–10 个，大流量泛标签（#AI #打工人）+ 精准标签（#开源项目 #AI工具）混配，已写进文案
- **互动预埋**：文案结尾是提问（"你的周报值几个 token？"）+ 晒卡引导——战报卡片本身会带来二次传播
- **数据合规**：工具内所有估算已带「娱乐向估算」标注，文案里不要说「真实额度」「官方数据」这类断言
- **发布后 1 小时**：守评论区，秒回前排评论（算法权重高）；置顶链接评论
- **如果被问「准不准」**：标准回复口径——「按各家官方 API 定价算的，每条数据都标了来源，订阅额度是社区估算，主打一个娱乐😄」
