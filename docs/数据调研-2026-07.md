## 各家要点与坑

- **OpenAI**：2026 年主力已是 GPT-5.6 三档（Sol $5/$30、Terra $2.5/$15、Luna $1/$6）。GPT-5.5 Pro $30/$180 是全场最贵，保留用于制造对比效果。官方页可直接抓取，更新最容易。
- **DeepSeek**：deepseek-chat / deepseek-reasoner 旧名 2026-07-24 弃用，已换成 V4 命名。V4 Pro 官方页直接标折后价 $0.435/$0.87。V4 Flash $0.14/$0.28 仍是全场最便宜的前沿模型。
- **Kimi**：国内平台按人民币计价（K2.6 ¥6.5/¥27），海外 api.moonshot.ai 是另一套 USD 价格，两边不等价。我们取国内价。
- **MiMo**：2026-05-27 永久降价后 ¥1/¥2（V2.5），刻意对齐 DeepSeek Flash 的美元价（$0.14/$0.28）。国内外双币官方均公示，数据质量最好的国产家。
- **GLM（最大的坑）**：官方 bigmodel.cn/pricing 为 JS 渲染无法抓取。第三方数据互相矛盾：一说 GLM-5 $1.5/$4.5（¥10.8/¥32.4），一说 GLM-5.1 ¥6/¥24 且 GLM-5 更便宜 1/3，还有说输出 $3.20。2026 年 GLM 走的是涨价路线（多篇报道印证）。**已取 $1.5/$4.5 标 estimate，上线前必须人工开浏览器核实。**
- **豆包**：火山引擎定价页 JS 渲染。豆包 2.1 Pro ¥6/¥30 来自证券时报发布报道，可信度中高。注意豆包全系按输入长度阶梯计价，我们只取最低档。
- **千问**：同样阶梯计价（Qwen3-Max ≤32K ¥2.5/¥10，32-128K ¥4/¥16）。注意百炼平台有 Qwen3.7-Max 上线的信息，旗舰可能已迭代，人工核实时确认最新旗舰名。
- **混元**：2026-03-11 官方涨价公告，HY2.0 Instruct 从 ¥0.8/¥2 涨到 ¥4.505/¥11.13（涨 5 倍+）。轻量 A13B ¥0.5/¥2 来自官方文档页（可抓取）。

## 订阅额度（plans.json）

- Claude：官方只说倍率（Max=5x/20x Pro），token 数是社区实测估算（Pro 约 30-50 万 input tok/5h 窗口）。2026-05-06 官方把 5 小时额度翻倍过一次——说明这类数据随时会变。
